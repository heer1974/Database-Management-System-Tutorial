# Database Management System Tutorial

## Overview
This project demonstrates the implementation and comparison of relational and NoSQL database systems using PostgreSQL and MongoDB.

## Technologies Used
- PostgreSQL
- MongoDB
- SQL
- NoSQL
- Database Design

## Features
- Database creation
- Table and collection design
- CRUD operations
- SQL JOIN queries
- Transactions (COMMIT & ROLLBACK)
- MongoDB document operations
- Database security concepts

- MongoDB implementation
